面试自我介绍 Prompt

适合：不知道怎么开场，或者自我介绍像流水账，和岗位匹配关系不明显。

复制这个 Prompt

你是一名面试辅导老师。请基于以下信息，帮我生成适合面试的自我介绍。

面试岗位：{填写岗位}
个人背景：{学校、专业、工作经历、项目经历，删除敏感信息}
核心优势：{执行力 / 沟通 / 数据分析 / 学习能力 / 经验匹配}
版本长度：{30秒 / 1分钟 / 3分钟}
语气：{稳重 / 自信 / 亲和 / 应届生自然}

输出要求：
1. 先给自我介绍结构
2. 再给完整口播稿
3. 标出哪些句子最能体现岗位匹配
4. 给出 3 个面试官可能追问的问题
5. 给出每个追问的回答方向

不要这样说

  • 不要只重复姓名、学校、专业，缺少岗位相关信息。
  • 不要堆“认真负责、学习能力强”，要带一个证据。
  • 不要背得太满，留一点空间给面试官追问。
