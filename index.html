这个站解决什么问题

很多人会用 AI，但不会把求职需求说清楚。这个站先不做复杂工具，只把常见求职场景整理成可复制的 Prompt：目标岗位、已有经历、修改目标、输出格式都写清楚。

第一版怎么验证

第一版只看搜索收录、自然访问和复制行为。有人搜索、有人打开、有人复制，才继续扩展更多岗位和模板。

边界说明

本站只提供写作辅助提示词，不保存用户简历，不承诺求职成功。使用时不要粘贴身份证、手机号、薪资、客户资料或公司机密。

首页示例：通用简历诊断 Prompt

复制后把括号里的内容换成自己的情况。不要粘贴敏感个人信息。

你是一名资深 HR 和简历顾问。请基于以下信息帮我做简历诊断：

目标岗位：{填写目标岗位}
工作年限：{应届 / 1-3年 / 3-5年 / 5年以上}
我的经历：{粘贴简历片段，删除手机号、身份证、薪资和公司机密}
优化目标：{表达更专业 / 突出成果 / 转行匹配 / 面试更好讲}

请按以下结构输出：
1. 简历最大的问题
2. 岗位匹配度分析
3. 可以量化的经历线索
4. 建议重写的 3 段表述
5. 面试时可以延展讲的亮点
6. 修改后的简历片段示例

这个站对应视频里的步骤

第一站先跑通流程，不追求功能复杂。

  1. 选题：AI 简历提示词，面向求职搜索需求。
