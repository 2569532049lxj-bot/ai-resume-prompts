PPT 大纲生成 Prompt

适合：老板要汇报、客户要方案、团队要培训，但你还没想清楚结构。

复制这个 Prompt

你是一名资深 PPT 策划顾问。请基于以下信息生成一份 PPT 大纲：

主题：{例如：季度复盘 / 竞品分析 / 新员工培训 / 项目方案}
使用场景：{汇报 / 培训 / 路演 / 方案 / 课程}
观众对象：{老板 / 客户 / 同事 / 学生}
页数：{5页 / 8页 / 10页 / 15页}
风格：{商务简洁 / 数据分析 / 故事型 / 说服型}

请按页输出：
- 页码
- 页面标题
- 核心观点
- 3 个内容要点
- 可视化建议
- 演讲备注

结构建议

  • 5 页：背景、问题、方案、执行、总结。
  • 8 页：封面、背景、目标、现状、问题、方案、计划、总结。
  • 10 页：封面、目录、背景、目标、现状、问题、方案、案例/数据、行动计划、总结。
