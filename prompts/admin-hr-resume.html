行政/人事简历怎么写得更专业

适合：简历里都是“负责日常行政、人事基础工作、员工沟通”，但缺少专业表达。

复制这个 Prompt

你是一名行政/人事岗位简历顾问。请帮我把事务性经历改成更专业、更有岗位匹配度的简历表达。

目标岗位：{行政专员 / 人事专员 / HR助理 / 办公室文员}
工作年限：{应届 / 1-3年 / 3-5年}
原始经历：{粘贴简历片段}
涉及模块：{招聘 / 入离职 / 培训 / 考勤 / 采购 / 会议 / 档案 / 员工服务}

请输出：
1. 这段经历体现了哪些能力
2. 哪些表述太口语或太泛
3. 可以补充的流程、频次、人数、周期、协作对象
4. 3 段更专业的简历改写
5. 一段适合面试展开的 STAR 故事

可补充的信息

  • 服务人数、招聘岗位数、入离职办理数量、培训场次。
  • 有没有优化表格、流程、通知模板、档案分类方式。
  • 和哪些部门协作，解决过哪些具体问题。
